# duckdbver

To install dependencies:

requires bun install

```bash
brew tap oven-sh/bun 
brew install bun
```

```bash
make install
```

To build the duckdver cli:

```bash
make build
```

The cli is put into bin\duckdbver

```bash
duckdbver path/to/data.duckdb
```
